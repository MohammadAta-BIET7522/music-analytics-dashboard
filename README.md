🎧 Music Analytics Platform
🚀 Overview

A production-style multi-page analytics dashboard built using Streamlit to analyze music traffic data, providing actionable insights into user engagement, geography, and content performance.

💡 Key Features
Multi-page dashboard architecture
KPI tracking (CTR, events, clicks)
Conversion funnel visualization
Geographic traffic analysis
Content performance insights
Business recommendations
🧠 Key Insights
Identified top-performing artists and regions
Calculated engagement metrics (CTR)
Built decision-support analytics
🛠 Tech Stack
Python
Pandas
Streamlit
Plotly
